# v2026.4.21

Release date: 2026-04-21

## Highlights

- Added a built-in GitHub destination-binding preset so `api.github.com`, GitHub web domains, GitHub Pages, GitHub assets, and GitHub raw/content domains can be pinned to a stable active node under transparent TUN mode.
- Tightened TUN privilege readiness checks so the WebPanel only reports sudo readiness when the installed helper matches the exact expected helper command.
- Fixed TUN helper DNS restoration on Linux systems using `systemd-resolved` and NetworkManager so stopping transparent mode restores the DNS scope instead of leaving the machine in a degraded state.
- Fixed helper and installer source resolution when WebPanel runs with an external config path, ensuring the installed helper and xray binary are compared against the running repository rather than unrelated config-directory scripts.
